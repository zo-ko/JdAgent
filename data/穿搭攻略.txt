一、基础搭配原则

1. 全身颜色控制在三种以内，避免过于杂乱
2. 上松下紧或上紧下松，避免整体过于臃肿或紧绷
3. 突出一个重点，其余保持简洁
4. 根据场合选择风格，不同场景穿搭不同

二、颜色搭配

1. 黑白灰属于基础色，百搭不出错
2. 同色系搭配更显高级（如深浅不同的蓝色）
3. 上浅下深更显稳重，上深下浅更显活泼
4. 避免高饱和颜色大面积混搭

三、身材修饰技巧

1. 个子矮：提高腰线（高腰裤、短上衣）
2. 个子高：可以尝试宽松或层次感穿搭
3. 偏瘦：选择有层次或略宽松的衣服
4. 偏胖：选择深色、直筒版型，避免紧身

四、常见万能搭配

1. 白T恤 + 牛仔裤 + 运动鞋
2. 衬衫 + 休闲裤 + 小白鞋
3. 卫衣 + 牛仔裤 + 板鞋
4. 西装外套 + T恤 + 西裤/牛仔裤

五、季节搭配建议

1. 春秋：外套（风衣、夹克）+ 内搭
2. 夏天：短袖 + 短裤或薄长裤，注重清爽
3. 冬天：大衣/羽绒服 + 分层穿搭（内搭+毛衣）

六、细节提升质感

1. 衣服尽量合身，版型比品牌更重要
2. 保持干净整洁，避免褶皱
3. 鞋子要干净，会明显提升整体形象
4. 合理搭配配饰（帽子、手表、包）

七、避免常见错误

1. 全身过于宽松或过于紧身
2. 颜色过多、过杂
3. 不考虑场合随意搭配
4. 忽视鞋子与整体风格的统一
